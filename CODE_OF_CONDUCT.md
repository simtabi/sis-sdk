# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1.

In short: be kind, be patient, and assume good faith. Harassment,
trolling, and personal attacks are not tolerated; sustained inappropriate
behavior may result in being asked to leave the project.

## Reporting

To report a Code of Conduct violation, email **opensource@simtabi.com**.
Reports are kept confidential within the maintainer team.

The full Contributor Covenant 2.1 text is available at
